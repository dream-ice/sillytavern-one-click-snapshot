# 一键快照

把角色/用户的多个版本直接嵌入酒馆原生编辑器，并把完整状态保存为可复用快照。

## 界面位置

- **角色版本**：角色编辑器的“更多…”菜单中，选择“管理版本…”。
- **用户版本**：Persona 页面“更多…”菜单中，选择“管理版本…”。
- **一键快照**：底部输入栏的魔法棒菜单中，点击“**一键快照**”打开悬浮窗。

原生描述框仍是唯一的编辑位置：在“管理版本…”悬浮窗中可应用、保存当前版本、新建、命名与删除版本。

## 快照内容

创建快照时可独立选择：

- 角色版本；
- 用户版本；
- 世界书（显示/保存全局、角色主、角色附加、聊天世界书的来源，以及每一条目的开关）；
- 预设（当前预设；OpenAI Chat Completion 还保存 Prompt Manager 每个条目的开关）。

快照窗中每张卡片可展开查看保存的世界书、条目和预设条目。快照可以：直接应用、绑定到当前聊天、或删除。

世界书范围可继续展开，分别选择全局、角色主、角色附加和聊天世界书；未选择的来源不会写入快照。快照分组和查看内容均可折叠，查看内容只显示启用条目。检测到 Preset Transfer 兼容分组数据时，世界书与预设条目也会按其分组显示。

## 聊天绑定

聊天元数据仅保存快照 ID 与“自动应用”开关。因此可保持绑定关系，但暂停进入该聊天时自动应用；之后可随时重新启用，不需要重新绑定。

## 注意

- 角色卡、世界书和预设是酒馆的共享文件。插件通过进入聊天时恢复快照，实现“同一角色在不同聊天使用不同版本”。不要同时让 Alternate Fields 管理同一批角色字段。
- 备份时请同时备份酒馆的 `settings.json`（版本库和快照库）与聊天文件（绑定关系）。
