# 一键快照

为 SillyTavern 保存和切换一组可复用的“状态组合”：角色/用户版本、世界书条目、预设提示词条目与正则规则。

快照记录的是**引用和开关状态**，不是旧内容的副本。之后即使你在酒馆面板里改了版本、世界书条目、提示词或正则的正文与名称，只要对应 ID 仍存在，应用快照时仍会作用于同一条内容，并保留它现在的内容。

## 安装

1. 在 SillyTavern 的扩展管理页选择“从 URL 安装扩展”。
2. 粘贴仓库地址：`https://github.com/dream-ice/sillytavern-one-click-snapshot`
3. 安装后刷新页面。

要求：SillyTavern `1.12.0` 或更高版本。插件入口位于底部快捷回复（Quick Replies）栏的“**一键快照**”按钮；请确保快捷回复栏已启用。

## 使用方式

### 1. 创建角色/用户版本

- **角色版本**：打开角色卡，进入“更多…” → “管理版本…”。
- **用户版本**：打开 Persona 页面，进入“更多…” → “管理版本…”。

先保存并应用一个版本，再创建包含角色或用户的快照。版本管理中保存当前正在使用的版本，会同步原生描述框；保存其他版本只更新版本本身，需要切换时再点击对应版本的“应用”。版本改名会自动同步到关联快照的显示名称；版本内容则以版本当前保存的内容为准。

版本管理页的“对比版本”可并排查看任意两个版本的描述差异：来源中删除的内容标红，目标中新加的内容标青。

### 2. 保存快照

打开“**一键快照**”，填写名称后按需勾选范围：

- 角色版本、用户版本；
- 世界书：全局、角色主/附加、用户绑定、聊天世界书，以及其中条目的开关；
- 预设：当前预设与 Prompt Manager 条目的开关；
- 正则：全局、角色局部、当前预设正则及其启用状态。

可使用分组整理快照，并通过“查看内容”确认将要应用的范围。

### 3. 应用或绑定

- **应用**：立刻应用快照。
- **绑定聊天**：进入该聊天时自动应用；可单独停用而不解绑。
- **绑定角色**：作为角色的默认快照，用于没有更具体绑定的聊天。
- **绑定开场白**：切换到指定主/备选开场白时自动应用对应快照。

自动应用优先级为：**聊天绑定 > 开场白绑定 > 角色默认绑定**。如快照包含用户版本，绑定时可选择仅应用兼容内容，或使用酒馆原生的用户绑定功能。

## 重要行为

- 快照不会把旧正文覆盖回世界书、预设提示词或正则脚本；它只恢复已记录条目的启用/停用状态。
- 角色与用户快照按版本 ID 应用，版本改名或修改内容不需要重存快照。
- 应用快照不会改写角色的主开场白或备选开场白目录。
- 正则状态有变化时会重载当前聊天，使规则立即生效；状态一致时不会重载。
- 如果某个版本、条目或脚本已被删除/搬迁而无法按原引用找到，插件会跳过该部分，不会新建或覆盖其他内容。

## 备份

快照库与版本库保存在 SillyTavern 的 `settings.json`；聊天/开场白绑定保存在对应聊天的元数据中。迁移或备份酒馆时，请同时备份它们。

## 兼容性

不要同时让其他扩展管理同一批角色字段或正则开关，以免相互覆盖。世界书和预设在酒馆中通常以名称/文件定位；整体改名或移动时，可能需要重新选择或更新对应快照。
